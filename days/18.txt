m{a-zA-Z}

Set mark and jump to it with `{a-zA-Z}
Lowercase letters creates marks local to the current file
Uppercase letters creates global marks.

Jumping to global mark will navigate to a different file

a-z marks are valid within one file and can be used with operators. For example
v`a will mark text from cursor to mark a.

A-Z marks (file marks) are valid between files.

Bonus: :marks will bring up a list of all marks.

Bonus: m` sets the previous context mark. This can be jumped to with ``

gi uses.
